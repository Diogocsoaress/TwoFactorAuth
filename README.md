Como executar a aplicação:

1.	Abrir o docker
2.	Abrir a pasta da aplicação num IDE à escolha;
3.	Executar no terminal da IDE: “docker build -t authentication-app .”;
4.	Executar no terminal da IDE: “docker-compose up --build”;
5.	Aceder ao url “localhost” num navegador à escolha.
6.	Efetuar o registo e ler o Código QR com o Google Authenticator;
7.	Fazer o login e inserir a OTP gerada pelo Google Authenticator;
